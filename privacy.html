MultipleBot Privacy Policy:

When using MultipleBot, I store data about your server to bring you a better experience (if you use certain features which are said in this privacy policy).

Last Modified October 9, 2023.

What information do I store from the people that use MultipleBot?

Only public information about your discord server. This is your Server ID, Channel ID(s).

Server ID and Channel ID(s) are used for the setmodlog (modlog) feature, without Server ID/Channel ID(s) MultipleBot cannot send logs of messages as Server ID and Channel ID is required to identify your discord server for such feature.

For any information sent via the setmodlog feature, the bot will attempt to remove personal information from the message (numbers). MultipleBot does not store message content in a database for storage and privacy purposes.

Anonymous Collection

I may collect the amount (in numbers) of discord servers which use this bot/service. This in no way can identify you or your discord server. This information may be used for Advertising or such.

How do I use your information?

Your information is used within the bot for features which require said information.

If you have a privacy concern, please contact:

me on discord https://discord.gg/YSEB7PnHVV and explain what the concern is.

Security Measures:

This only applies if modlogs were set up. The data collected is stored in MongoDB with a very long password. Only the owner has access to it directly.

Agreement

By using MultipleBot, you agree to these terms of this Privacy Policy. If you do not agree to these terms, you must not use this service.

You may delete the data the bot has collected about your server if the bot is in your server and you have proper permissions by using this command: /deldata and/or kicking the bot from your server while the bot is online.
